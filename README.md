# Next.js Boilerplate

## How to use?
```sh
$ git clone https://github.com/wonism/my-next-boilerplate <<YOUR_PROJECT_NAME>>
$ cd <<YOUR_PROJECT_NAME>>
$ rm -rf .git
```

## Features
- state management with `recoil`
- css in js with `emotion`
- `typescript`
- `husky`, `lint-staged`, `eslint`, `prettier`
